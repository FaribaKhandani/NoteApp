## About The Project



This is simple Note Appication which used the latest technologies and libraries; 


Which technology:
* Clean architecture which is SOLID principles and MVVM 
* Room, Navigation, Dependency Injection



<p align="right">(<a href="#top">back to top</a>)</p>



### Modules


* [Kotlin](https://github.com/JetBrains/kotlin)
* [MVVM](https://developer.android.com/jetpack/guide)
* [Room](https://developer.android.com/training/data-storage/room)
* [Dagger](https://developer.android.com/codelabs/android-dagger?hl=ko#0)

## Getting Started


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/FaribaKhandani/NoteApp.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
